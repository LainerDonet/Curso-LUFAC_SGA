Configuración de la hora a través de timedatectl

El comando timedatectl en Linux se utiliza para consultar y cambiar la configuración del reloj del sistema y habilitar o deshabilitar los servicios de sincronización de tiempo. Aquí tienes algunos ejemplos de cómo usarlo:

  1. Ver la hora actual del reloj del sistema:

    timedatectl

  2. Establecer directamente la hora local del reloj del sistema (reemplaza "yyyy-MM-dd hh:mm:ss" con la hora deseada):

    timedatectl set-time "yyyy-MM-dd hh:mm:ss"

  3. Listar las zonas horarias disponibles:

    timedatectl list-timezones

  4. Establecer la zona horaria del sistema (reemplaza "America/Mexico_City" con la zona deseada):

    timedatectl set-timezone "America/Mexico_City"

  5. Habilitar la sincronización con el Protocolo de Tiempo de Red (NTP):

    timedatectl set-ntp true
