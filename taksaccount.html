Gestion de cuentas

La gestión de cuentas de usuario en Linux es fundamental para mantener la seguridad y la eficiencia en un entorno GNU/Linux. Aquí tienes algunos conceptos clave y comandos para manejar usuarios y grupos:

  1. Tipos de usuarios en LUFAC SGA:
    • Usuario root: El superusuario con privilegios administrativos completos.
    • Usuarios normales: Realizan tareas regulares con permisos limitados.
    • Usuarios del sistema: Asociados a servicios y aplicaciones, no usuarios reales.
  2. Grupos en Linux:
    • Grupo wheel/sudo: Contiene usuarios autorizados para ejecutar comandos como superusuario.
    • Grupo users: Agrupa a usuarios normales.
    • Grupo adm: Gestiona acceso a registros y archivos de registro.
  3. Comandos para la gestión de usuarios y grupos:
    • useradd: Añade un nuevo usuario al sistema.
    • passwd: Establece o cambia la contraseña de un usuario.
    • usermod: Modifica propiedades de usuario (nombre completo, grupo primario).
    • userdel: Elimina un usuario del sistema.
