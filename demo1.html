Demo Establecer Hostname

El comando hostname y hostnamectl en Linux se utilizan para gestionar y cambiar el nombre de host de un sistema. Esta es una breve descripción de cada uno:

  1. hostname:
    • El comando hostname permite ver o cambiar el nombre de host temporalmente en tiempo real.
    • Para cambiar el nombre de host de forma temporal, ejecuta:
      • sudo hostname nuevo-nombre-de-host
    • Esto afectará el nombre de host solo hasta que reinicies el sistema
  2. hostnamectl:
    • hostnamectl es parte de systemd y se utiliza para gestionar y cambiar el nombre de host de manera permanente.
    • Para cambiar el nombre de host de forma permanente, ejecuta:
      • sudo hostnamectl set-hostname nuevo-nombre-de-host
    • Este cambio se mantendrá incluso después de reiniciar el sistema

Ambos comandos necesitan permiso de administración para ser ejecutados. En LUFAC SGA los permisos de administración son requeridos por la herramienta para ejecutar estas operaciones y de no tenerlos no aparece la opciones.
